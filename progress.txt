================================================================================
PROJECT SCOPE
================================================================================

Split.AI is a workout split analysis tool that calculates net weekly stimulus
per muscle region using research-backed models. The system accounts for:
- Diminishing returns (Schoenfeld/Pelland marginal curves)
- Recovery windows and atrophy
- Systemic fatigue (CNS, axial/spinal, bilateral deficit)
- Motor unit recruitment efficiency (leverage matching)
- Tiered muscle involvement (prime/secondary/tertiary movers)

Target deliverables:
1. Backend engine (Python/FastAPI) - analyze splits, classify exercises
2. Database layer (Supabase) - user accounts, saved splits, workout logs
3. Frontend (Next.js/React) - modeled after tracked.gg
4. CLI tools - CSV/Excel import for split analysis
